# Hackathon app

Small Codex Sites app, scaffolded for rapid iteration.

## Project layout

- `app/` — site routes and page-level UI
- `components/` — reusable interface components
- `lib/` — shared helpers and application logic
- `public/` — static assets
- `tests/` — automated tests
- `docs/` — product and implementation notes

The app implementation will be added with the Codex Sites starter when the
product direction is ready.
